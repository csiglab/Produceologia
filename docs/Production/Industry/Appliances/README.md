# Electrical Appliances

> The **electrical appliances industry** produces household and small commercial devices that use electricity for daily tasks such as cooking, cleaning, heating, cooling, and personal care. This is a **light manufacturing industry** closely linked to consumer electronics, electrical equipment, and component manufacturing.
