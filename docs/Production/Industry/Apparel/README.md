# **Apparel Manufacturing**

> The **apparel (garment) industry** transforms textile products into finished clothing and apparel items for consumers. It includes the production of garments for men, women, and children, as well as undergarments, hosiery, and apparel accessories. Apparel manufacturing is a **light industry** that relies on upstream textile production and downstream retail distribution.
