# Pulp Industry

> The Pulp Industry produces cellulose pulp from wood, recycled paper, or other fibrous materials. Pulp is the primary raw material for paper, cardboard, tissue, and some specialty cellulose products. This industry is considered upstream in the broader Pulp and Paper sector.

## Product Space

| Product                      | Description                                                                      | NAICS Code      |
| ---------------------------- | -------------------------------------------------------------------------------- | --------------- |
| Wood Pulp                    | Pulp produced from wood through chemical, mechanical, or semi-chemical processes | 322110          |
| Recycled Paper Pulp          | Pulp produced from post-consumer or post-industrial paper                        | 322121          |
| Kraft Pulp                   | High-strength pulp produced using the kraft process                              | 322110          |
| Sulfite Pulp                 | Pulp produced using the sulfite chemical process                                 | 322110          |
| Specialty Cellulose Products | High-purity cellulose for industrial or chemical applications                    | 325199 / 322110 |
