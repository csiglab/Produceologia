# Agricultural Production: A Generic Notion

## Terreno Físico

> Terminología utilizada para referirse a la **unidad física base** de la producción agrícola, ganadera y forestal.

| **Categoría** | **Término** | **Definición** |
|--------------|-------------|----------------|
| General | Terreno agrícola | Superficie destinada a actividades agrícolas, sin especificar cultivo. |
| General | Parcela | Unidad mínima de tierra delimitada e identificable. |
| General | Predio | Propiedad rústica o rural, con o sin infraestructura. |
| General | Finca | Propiedad rural activa dedicada a producción agraria o mixta. |
| General – Organización productiva | Explotación | Conjunto de tierras, instalaciones y factores usados para producir. |
| General – Organización productiva | Unidad de producción | Unidad operativa donde se articulan los recursos productivos. |
| Tradicional – América | Hacienda | Propiedad agrícola o ganadera de gran tamaño, tradicional. |
| Tradicional – Mixto | Granja | Unidad dedicada a producción hortícola, avícola, lechera o mixta. |
| Legal – Catastro | Parcela catastral | Unidad gráfica registrada por el Catastro. |
| Legal – Suelo rústico | Parcela rústica | Parcela ubicada en suelo no urbanizable. |
| Tradicional – Andalucía | Cortijo | Finca agroganadera tradicional con vivienda y anexos. |
| Tradicional – Cataluña/Aragón | Masía | Casa rural tradicional con tierras agrícolas o forestales. |
| Tradicional – Histórico | Heredad | Propiedad con derechos históricos de riego u otros aprovechamientos. |
| Cultivos – Herbáceos | Campo de cultivo | Terreno destinado a cultivos anuales o rotaciones. |
| Cultivos – Herbáceos | Campiña | Territorio agrícola amplio dominado por cultivos extensivos. |
| Cultivos – Herbáceos | Cerealera | Terreno especializado en cereales. |
| Cultivos – Hortícolas | Huerta | Terreno fértil e intensivo de hortalizas en regadío. |
| Cultivos – Hortícolas | Vega | Terreno fértil en riberas o valles. |
| Cultivos – Forrajeros | Pradera | Terreno con vegetación para pasto o siega. |
| Cultivos – Forrajeros | Pastizal | Área natural o seminatural de producción de pasto. |
| Cultivos – Leñosos | Viñedo | Terreno dedicado al cultivo de vid. |
| Cultivos – Leñosos | Olivar | Terreno dedicado al cultivo del olivo. |
| Cultivos – Leñosos | Frutal | Parcela con árboles frutales. |
| Cultivos – Leñosos | Almendral | Terreno con almendros. |
| Cultivos – Leñosos | Nogaledo | Terreno con nogales. |
| Cultivos – Leñosos | Avellanal | Terreno con avellanos. |
| Cultivos – Tropicales | Bananal / platanal | Terreno destinado al cultivo de plátano o banana. |
| Cultivos – Tropicales | Cacaotal | Plantación de cacao. |
| Cultivos – Tropicales | Cafetal | Plantación de café. |
| Cultivos – Tropicales | Cañaveral | Terreno de cultivo de caña de azúcar. |
| Sistema de producción – Regadío | Tierras de riego | Terrenos con dotación de agua mediante sistemas de irrigación. |
| Sistema de producción – Regadío | Invernadero | Estructura cerrada para producción intensiva. |
| Sistema de producción – Regadío | Bajo plástico | Cultivo bajo cubierta plástica. |
| Sistema de producción – Secano | Secano | Terreno alimentado únicamente por la lluvia. |
| Sistema de producción – Secano | Erial | Tierras sin cultivar, recuperables para producción. |
| Sistema de producción – Intensivo | Malla sombra | Cultivo protegido con malla para controlar radiación. |
| Sistema de producción – Extensivo | Dehesa | Sistema agrosilvopastoral de baja intensidad (España). |
| Sistema de producción – Extensivo | Pastizal extensivo | Terrenos amplios de pastoreo. |
| Ganadero – General | Prado | Terreno de pasto natural o mejorado. |
| Ganadero – Agroganadero | Cortijo (mixto) | Finca donde coexisten agricultura y ganadería. |
| Ganadero – Trashumancia | Majada | Lugar donde se refugia o pernocta el ganado. |
| Forestal – Productivo | Pinar | Terreno de pinos para madera o resina. |
| Forestal – Productivo | Robledal | Terreno con robles para usos forestales. |
| Forestal – Productivo | Encinar | Bosque de encinas con aprovechamientos productivos. |
| Forestal – Productivo | Eucaliptal | Terreno de eucaliptos para madera o pasta. |
| Forestal – Mixto | Bosque productivo | Bosque manejado para producción sostenible. |
