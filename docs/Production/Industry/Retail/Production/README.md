# Production

> How does the retail industry operate, and what defines the characteristics of a retailer?

## Characterization

| **Dimension**                    | **Description**                                                                                                                                                                                                                                           |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Product Category**             | The types of goods or services offered, e.g., groceries, apparel, electronics, home goods.                                                                                                                                                                          |
| **Selling Vehicle**              | The channel(s) through which sales occur: **Offline**, **Online**, or **Omnichannel**.                                                                                                                                                                              |
| **Customer Interaction**         | The mode of engagement with consumers: **Direct-to-Consumer (D2C)**, **B2B**, or hybrid.                                                                                                                                                                            |
| **Stocking (Fulfillment Model)** | The inventory and fulfillment approach: <br>• **Buy-to-Sell** (traditional inventory) <br>• **Just-in-Time (JIT)** <br>• **Direct Fulfillment / Drop Shipping** <br>• **Consignment** <br>• **Subscription / Recurrent Delivery** <br>• **Display-Only / Showroom** |
