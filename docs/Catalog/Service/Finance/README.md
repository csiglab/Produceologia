## Product Space

| Industry                      | Product                  | Description                                                  | Tags                                         |
| ----------------------------- | ------------------------ | ------------------------------------------------------------ | ------------------------------------------- |

|                               | Savings Account          | Interest-bearing account for individuals                     | Service, Deposit, Interest                  |
|                               | Personal Loan            | Unsecured or secured loan for personal consumption          | Service, Credit, Loan                        |
|                               | Mortgage                 | Loan for purchasing residential or commercial property      | Service, Credit, Loan, Secured              |
|                               | Credit Card              | Revolving credit line for consumer purchases                | Service, Credit, Revolving                  |
| **Commercial Banking**        | Business Loan            | Loan for business financing needs                            | Service, Credit, Business                   |
|                               | Treasury Services        | Cash management, payments, and liquidity services for firms | Service, Cash Management, B2B               |
| **Investment Banking**        | M&A Advisory             | Strategic advisory for mergers and acquisitions             | Service, Advisory, Corporate                |
|                               | Underwriting             | Issuing and guaranteeing securities                          | Service, Capital Markets, Securities        |
| **Insurance**                 | Life Insurance           | Provides payout on death or after a set period              | Service, Risk, Life                          |
|                               | Auto Insurance           | Coverage for vehicle-related losses or damages              | Service, Risk, Property                     |
|                               | Health Insurance         | Coverage for medical and healthcare expenses                | Service, Risk, Health                       |
|                               | Reinsurance              | Insurance purchased by insurers to mitigate risk            | Service, Risk, B2B                           |
| **Investments**               | Seed Capital             | Early-stage funding for startups to develop ideas           | Service, Investment, Early-Stage, Startup  |

|                               | Pension Fund             | Long-term institutional investment pool for retirement      | Service, Investment, Institutional          |
|                               | Robo-Advisor             | Algorithm-driven portfolio management                        | Service, Investment, Digital, Fintech       |
| **Fintech**                   | Mobile Wallet            | Digital wallet for payments and money transfers             | Service, Digital, Payment                    |
|                               | P2P Lending              | Online platform connecting lenders and borrowers            | Service, Credit, Digital, Fintech           |
|                               | Cryptocurrency Exchange  | Platform for buying, selling, and trading digital assets    | Service, Digital, Investment                |
|                               | Neobank                  | Fully digital banking services                               | Service, Digital, Retail Banking, Fintech   |
