# Reflection

- How can I critically examine my own actions to ensure they align with the Lab’s mission?

- How can I evaluate my own thinking and decision-making in the context of advancing the Lab’s goals?

- How can I systematically learn from my experiences to improve future alignment with the Lab’s purpose?

- In what ways can I identify biases, assumptions, or patterns in my actions and thought processes that affect my contribution to the Lab’s mission?

## Guide

> This guide provides the fundamental elements for reflection using well-structured questions. When you feel confident with your answer, move it to History, along with a brief version of the response.
